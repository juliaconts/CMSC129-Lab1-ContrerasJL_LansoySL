## lostnfound

Project by: Julia Contreras & Sam Lansoy

A community-driven lost and found platform that helps people recover lost items and reunite found ones with their owners. Post what you've lost or found, add photos and location details, and let your community help reconnect what's missing.

## Website Features

1. Post lost or found items with title, description, location, and photo.
2. Image upload with automatic compression.
3. Soft delete with 30-day recovery window (Recently Deleted).
4. Restore or permanently delete posts.
5. Edit your existing posts.
6. Firebase Authentication (email/password).
7. Responsive layout with drawer navigation.
8. Toast notification for all actions.

## Tech Stack

**FERN** (Firebase, Express, React, Node)
